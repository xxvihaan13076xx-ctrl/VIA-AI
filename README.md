## 🎯 What is VIA AI?

VIA AI is a **boredom elimination app** built as a single HTML file. No frameworks, no installs, no accounts needed. Open it and go.

It gives you missions based on your interests, tracks your progress with an XP and level system, challenges you with anime games, teaches you Japanese, lets you build workout routines, cook with AI recipes, and much more — all in one place.

## 📱 Install as App

VIA AI is a **Progressive Web App** — install it like a real app on any device.

**Android (Chrome)**
1. Open the site in Chrome
2. Tap the **"Install VIA AI"** banner at the bottom
3. Tap **Install** ✅

**iPhone / iPad (Safari)**
1. Open in Safari
2. Tap the **Share button** (□↑)
3. Tap **"Add to Home Screen"** → **Add** ✅

**Windows / Mac / Linux (Chrome)**
1. Click the **install icon** in the address bar
2. Click **Install** ✅

> Works **offline** after first install thanks to Service Worker caching.

---

## 💾 Save & Restore Progress

Progress saves automatically to your device. To back up or move to a new device:

1. Open **☰ Menu → Save Data**
2. Tap **Download Save File** — saves a `.json` file
3. On the new device, go to **Save Data → Upload Save File**
